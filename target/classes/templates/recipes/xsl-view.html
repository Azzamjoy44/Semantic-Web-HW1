How This Works

The server loads recipes.xml, applies xsl/recipes.xsl via javax.xml.transform.Transformer, and passes userSkillLevel as an XSL parameter. The XSL template assigns CSS classes skill-match (yellow) or skill-other (green) based on primaryDifficulty = $userSkillLevel.
